Utilisation du Système 📋

  1. Accéder au système : http://localhost/gestion-tickets.php
  2. Voir les détails : Cliquer sur le bouton "👁️" dans la colonne Actions
  3. Imprimer un ticket : Cliquer sur le bouton "🖨️" pour impression thermique
  4. Filtrer les résultats : Utiliser les filtres en haut de page
  5. Rechercher : Utiliser la barre de recherche DataTables

Structure des Tables Principales 🗄️

  • CMD_VENTE : Table principale des tickets/commandes
  • CMD_VENTE_DETAIL : Détails des articles de chaque ticket
  • PARAMETRECAISSE : Configuration de la caisse (en-têtes, pieds de page)
  • CLIENT : Informations clients
  • ARTICLE : Catalogue des produits
